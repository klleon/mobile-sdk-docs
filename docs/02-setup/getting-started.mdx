# 시작하기

Klleon Chat SDK를 사용하여 웹 애플리케이션에 아바타 채팅 기능을 통합하는 것은 매우 간단합니다. 이 가이드에서는 SDK를 사용하기 위한 사전 준비사항부터 설치, 그리고 기본적인 초기화 방법까지 단계별로 안내합니다.

## 사전 준비 사항

Klleon Chat SDK를 사용하기 전에 다음 준비가 필요합니다.

1.  **클레온 스튜디오 계정**: SDK 기능을 사용하려면 [클레온 스튜디오](https://www.studio.klleon.io/) 계정이 필요합니다. 계정이 없으시다면 먼저 가입을 진행해 주세요.
2.  **도메인 등록**: SDK를 사용할 웹사이트의 [도메인](https://www.studio.klleon.io/sdk)을 클레온 스튜디오에 등록해야 합니다.
    - 로그인 후 'SDK 관리' 메뉴에서 등록할 수 있습니다.
    - 도메인 등록 과정에 어려움이 있다면 `partnership@klleon.io`로 문의해 주세요. (이메일 주소는 기존 문서 참고)
3.  **SDK 키 발급**: 도메인 등록 후, 해당 도메인에서 사용할 고유한 `sdk_key`가 발급됩니다. 이 키는 SDK 초기화 시 필요합니다.
4.  **아바타 ID 확인**: 사용하고자 하는 아바타의 고유 `avatar_id`를 확인해야 합니다. 이 ID 또한 SDK 초기화 시 필요합니다. 클레온 스튜디오의 [(아바타 목록)](https://www.studio.klleon.io/avatar) 페이지에서 사용 가능한 아바타를 확인할 수 있으며, **이 페이지에서 ID 복사가 가능한 아바타만 SDK 연동에 사용할 수 있습니다.**

## SDK 설치

Klleon Chat SDK는 웹 페이지에 `<script>` 태그를 추가하는 방식으로 간편하게 설치할 수 있습니다.

HTML 파일의 `<head>` 태그에 다음 스크립트 태그를 추가하세요.

```html title="index.html (SDK 설치)"
<!DOCTYPE html>
<html lang="ko">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>Klleon Chat SDK 시작하기</title>
    {/* Klleon Chat SDK 스크립트 추가 ({VERSION}을 실제 버전으로 변경) */}
    <script src="https://web.sdk.klleon.io/{VERSION}/klleon-chat.umd.js"></script>
  </head>
  <body>
    {/* 애플리케이션 콘텐츠 */}
  </body>
</html>
```

:::tip SDK 버전 확인
`{VERSION}` 부분에는 `X.Y.Z` 형식의 실제 SDK 버전을 입력해야 합니다. (예: `1.0.0`)
사용 가능한 최신 SDK 버전은 클레온 스튜디오의 SDK 관리 페이지 또는 별도의 공지 채널을 통해 확인하실 수 있습니다. 항상 최신 버전을 사용하는 것을 권장하지만, 특정 버전 사용이 필요한 경우 해당 버전을 명시해주세요.
:::

## SDK 초기화

SDK 스크립트가 로드된 후, `window.KlleonChat.init()` 메서드를 호출하여 SDK를 초기화할 수 있습니다. 초기화는 일반적으로 DOM이 준비된 후 또는 메인 애플리케이션 스크립트 내에서 수행합니다.

가장 기본적인 초기화에는 `sdk_key`와 `avatar_id` 두 가지 옵션이 필수로 요구됩니다. (이벤트 리스너 등록 후 `init`을 호출하는 것을 권장합니다.)

```javascript title="main.js 또는 HTML 내 <script> 태그 (초기화 예시)"
// DOM이 로드된 후 SDK 관련 로직 실행
document.addEventListener("DOMContentLoaded", async () => {
  if (window.KlleonChat) {
    // 1. Status 이벤트 리스너 등록
    window.KlleonChat.onStatusEvent((status) => {
      console.log("SDK Status Event:", status);
      // 상세한 Status 값과 의미는 '이벤트 처리' 문서를 참고하세요.
      if (status === "VIDEO_CAN_PLAY") {
        console.log("아바타 영상 재생 준비 완료!");
      }
    });

    // 2. Chat 이벤트 리스너 등록
    window.KlleonChat.onChatEvent((chatData) => {
      console.log("SDK Chat Event:", chatData);
      // 상세한 ChatData 구조와 chat_type은 '이벤트 처리' 문서를 참고하세요.
    });

    // 3. SDK 초기화
    try {
      await window.KlleonChat.init({
        sdk_key: "YOUR_SDK_KEY", // 발급받은 SDK 키로 교체하세요.
        avatar_id: "YOUR_AVATAR_ID", // 사용할 아바타 ID로 교체하세요.
      });
      console.log("Klleon Chat SDK 초기화 시도 완료.");
    } catch (error) {
      console.error("Klleon Chat SDK 초기화 실패:", error);
    }
  } else {
    console.error("Klleon Chat SDK가 로드되지 않았습니다.");
  }
});
```

## 바로 따라하기: 전체 예제

다음은 Klleon Chat SDK를 웹 페이지에 설치하고, 이벤트를 등록하며, 초기화하는 가장 기본적인 전체 HTML 예제입니다. 이 코드를 `index.html` 파일로 저장하고 웹 브라우저에서 열어보세요. (실제 동작을 위해서는 `YOUR_SDK_KEY`와 `YOUR_AVATAR_ID`를 유효한 값으로 변경해야 합니다.)

```html title="index.html (전체 예제)"
<!DOCTYPE html>
<html lang="ko">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>Klleon Chat SDK 시작하기</title>
    {/* 1. Klleon Chat SDK 스크립트 추가 ({VERSION}을 실제 버전으로 변경) */}
    <script src="https://web.sdk.klleon.io/{VERSION}/klleon-chat.umd.js"></script>
  </head>
  <body>
    <h1>Klleon Chat SDK 데모</h1>

    <div class="container">
      <avatar-container
        class="avatar-container"
        volume="100"
        videoStyle='{"borderRadius": "24px"}'
      ></avatar-container>
      <chat-container
        delay="30"
        type="text"
        isShowCount="true"
        class="chat-container"
      ></chat-container>
    </div>

    {/* 간단한 UI 레이아웃을 위한 스타일 */}
    <style>
      body {
        display: flex;
        flex-direction: column;
        align-items: center;
        font-family: sans-serif;
        padding: 20px;
        margin: 0;
      }
      .container {
        display: flex;
        width: 800px;
        max-width: 100%;
        height: 600px;
        gap: 20px;
        margin-top: 20px;
        border: 1px solid #ddd;
        padding: 10px;
        box-sizing: border-box;
      }
      .avatar-container {
        flex: 1;
        border: 1px solid #ccc;
        background-color: #f0f0f0;
      }
      .chat-container {
        flex: 1;
        border: 1px solid #ccc;
        border-radius: 8px;
      }
    </style>

    <script>
      // 3. SDK 초기화 및 이벤트 리스너 등록
      document.addEventListener("DOMContentLoaded", async () => {
        if (window.KlleonChat) {
          console.log("Klleon Chat SDK 로드 확인됨.");

          // Status 이벤트 리스너 등록
          window.KlleonChat.onStatusEvent((status) => {
            console.log("Status Event Received:", status);
            // 상세한 Status 값과 의미는 '이벤트 처리' 문서를 참고하세요.
            if (status === "VIDEO_CAN_PLAY") {
              console.log(
                ">>> 아바타 영상 재생 준비 완료! SDK 메서드 사용 가능 상태입니다."
              );
            }
          });
          console.log("onStatusEvent 리스너 등록됨.");

          // Chat 이벤트 리스너 등록
          window.KlleonChat.onChatEvent((chatData) => {
            console.log("Chat Event Received:", chatData);
            // 상세한 ChatData 구조와 chat_type은 '이벤트 처리' 문서를 참고하세요.
          });
          console.log("onChatEvent 리스너 등록됨.");

          // SDK 초기화
          try {
            await window.KlleonChat.init({
              sdk_key: "YOUR_SDK_KEY", // 실제 SDK 키로 변경하세요.
              avatar_id: "YOUR_AVATAR_ID", // 실제 아바타 ID로 변경하세요.
            });
            console.log("KlleonChat.init() 호출 완료.");
          } catch (error) {
            console.error("Klleon Chat SDK 초기화 중 오류 발생:", error);
          }
        } else {
          console.error("Klleon Chat SDK가 로드되지 않았습니다.");
        }
      });
    </script>
  </body>
</html>
```
