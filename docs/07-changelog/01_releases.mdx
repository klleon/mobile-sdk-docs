# 릴리스 노트

Klleon Ondevice SDK의 버전별 변경사항입니다.

---

## v0.2.17 (Latest)

**릴리스일**: 2026년 3월

### 버그 수정
- 일부 디바이스에서 오디오 입력 스트림 열기 실패 수정
- `play()` 함수에 `mmapEnabled` 파라미터 추가

### 개선사항
