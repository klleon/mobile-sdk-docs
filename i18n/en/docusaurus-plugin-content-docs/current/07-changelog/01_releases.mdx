# Release Notes

Version-specific changes for the Klleon Ondevice SDK.

---

## v0.3.6 (Latest)

**Release Date**: April 2026

### Changes
- Added `avatarId` parameter to `ResourceManager.init()`
- YUV buffer lifecycle separation: buffer reuse for same avatar re-entry (0ms)
- 3-second cooldown when switching avatars (OOM defense for low-end TV SoCs)
- Buffer retained on `finish()` (`resetState`) + full release on app exit (`deinit`)
- Automatic background memory release based on `onTrimMemory` / `onLowMemory`
- IO coroutine transition inside `init()` (main thread blocking removed)

---

## v0.3.5

**Release Date**: April 2026

### Changes
- RAW YUV video decoding pipeline implementation (MP4→YUV pre-conversion)
- NV12/I420 color format auto-detection and conversion
- Activity lifecycle-based native memory release

## v0.3.3

**Release Date**: March 2026

### Changes
- Pre-computed .style file loading (StyleFileLoader added)

- WebSocket voice signals added (SPEECH_START, SPEECH_END, STOP_RESPONSE_MESSAGE)
- ResourceManager .style file download support
