# Release Notes

Version-specific changelogs for Klleon Ondevice SDK.

---

## v0.3.6 (Latest)

**Release Date**: April 2026

### Changes
- Added `avatarId` parameter to `ResourceManager.init()`
- Separated YUV buffer lifecycle: buffer reuse on same avatar re-entry (0ms)
- 3-second cooldown on avatar switching (OOM protection for low-end TV SoCs)
- Buffer retained on `finish()` (`resetState`) + full release on app exit (`deinit`)
- Automatic background memory release based on `onTrimMemory` / `onLowMemory`
- Switched `init()` internals to IO coroutine (eliminated Main thread blocking)

---
