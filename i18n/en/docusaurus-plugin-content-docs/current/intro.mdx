# Intro

# Klleon Ondevice SDK

[![Version](https://img.shields.io/badge/version-0.2.3-blue.svg)](https://github.com/anthropics/klleon-ondevice-sdk)

Klleon Ondevice SDK is a powerful SDK that provides on-device AI capabilities along with video processing and real-time messaging. With this SDK, you can easily integrate chat functionality into your applications that enables real-time interaction with AI-powered avatars.

## Key Features

### Video Processing
- AI video playback
- LLM-based TTS
- High-performance on-device AI processing

### Real-time Messaging
- WebSocket-based real-time communication
- Message sending and echo functionality
- State management
