# Intro

# Klleon Ondevice SDK

Klleon Ondevice SDKは、ビデオ処理、リアルタイムメッセージングとともにオンデバイスAI機能を提供する強力SDKです。このSDKを使用することで、AIベースのアバターとリアルタイムでコミュニケーションするチャット機能をアプリケーションに簡単に統合できます。

## 主要機能

### ビデオ処理
- AIビデオ再生
- LLMベースのTTS
- 高性能オンデバイスAI処理

### リアルタイムメッセージング
- WebSocketベースのリアルタイム通信
- メッセージ送信およびエコー機能
- 状態管理
