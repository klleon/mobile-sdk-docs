# Intro

# Klleon Ondevice SDK

![Version](https://img.shields.io/badge/version-0.5.0-blue.svg)

Klleon Ondevice SDKは、ビデオ処理、リアルタイムメッセージングとともにオンデバイスAI機能を提供する強力なSDKです。このSDKを使用することで、AIベースのアバターとリアルタイムでコミュニケーションするチャット機能をアプリケーションに簡単に統合できます。

## 主な機能

### ビデオ処理
- AIビデオ再生およびリップシンク
- 高性能オンデバイスAI処理
- ビデオの一時停止/再開制御

### リアルタイムメッセージング
- WebSocketベースのリアルタイム通信
- メッセージ送信、エコー、発話中断機能
- 状態管理

### 音声認識
- 低遅延マイク録音
- タップして話す方式の音声認識
- USBマイクの自動検出および切り替え

### Android TVサポート
- 270度回転レンダリング
- TVリモコンキーマッピング
- TVフォーカスUI最適化
