# 設定

## 1) 要件

- **SDKバージョン**: 0.2.17
- **最小SDK**: 26 (Android 8.0)
- **ターゲットSDK**: 36
- **対応ABI**: arm64-v8a, armeabi-v7a

---

### リソース

SDKにはビデオファイル（`.mp4`）と顔モデルファイル（`.npz`）が必要です。
これらのファイルを`assets`（`src/main/assets`）フォルダに配置した後、`ResourceManager.download()`を呼び出してキャッシュディレクトリにコピーします。

```kotlin
import io.klleon.ondevice.ResourceManager

val resourceManager = ResourceManager()

resourceManager.download(
    context = this,
    mp4FileName = "loop_sample.mp4",
    npzFileName = "loop_sample.npz",
    onSuccess = {
        // リソース準備完了 — play()の呼び出しが可能です
        // resourceManager.vpPath, resourceManager.fpPath が自動設定されます
    },
    onError = { error ->
        Log.e("Setup", "リソース準備失敗", error)
    }
)
```

:::info
`download()` 完了後、`vpPath`（ビデオパス）と`fpPath`（顔モデルパス）が自動設定されます。
`play()` を呼び出す前に、必ず`download()`が完了している必要があります。
:::

## 2) Android構成

```kotlin
android {
    compileSdk = 36

    defaultConfig {
        minSdk = 26
        targetSdk = 36
    }

    packaging {
        jniLibs {
            pickFirsts += "**/libc++_shared.so"
        }
    }
}
```

## 3) Gradle依存関係

アプリレベルの`build.gradle.kts`に以下の依存関係を追加します：

```kotlin
dependencies {
    // SDK
    implementation("io.klleon:ondevice-sdk:0.2.17")

    // Coroutines（SDK必須）
    implementation("org.jetbrains.kotlinx:kotlinx-coroutines-core:1.9.0")
    implementation("org.jetbrains.kotlinx:kotlinx-coroutines-android:1.9.0")
}
```

:::info バンドルライブラリ
SDK AARには以下のライブラリが含まれています：
- OkHttp, Okio, Retrofit
- Jackson (Core, Module)
- TensorFlow Lite
- OpenCV
- PyTorch Mobile
- Oboe
- ONNX Runtime
- Agora RTC

アプリでOkHttpまたはOkioを使用するライブラリ（例：Coil）がある場合、バージョンの競合が発生する可能性があります。この場合、`RuntimeClasspath`から該当グループをexcludeしてください：
```kotlin
configurations.matching { it.name.contains("RuntimeClasspath", ignoreCase = true) }.configureEach {
    exclude(group = "com.squareup.okhttp3")
    exclude(group = "com.squareup.okio")
}
```
:::

## 4) ProGuard / R8

SDK AARに必要なProGuard/R8ルールが含まれているため、**別途の設定は不要です。**

`UnsatisfiedLinkError`が発生した場合は、上記Android構成の`jniLibs.pickFirsts`設定を確認してください。
